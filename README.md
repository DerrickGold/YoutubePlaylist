# YoutubePlaylist

A plugin for the [Smart Reflect Server] (https://github.com/DerrickGold/SmartReflectServer) software that
allows one to create, save, edit, and play youtube playlists on a smart mirror display.

##Installation:
* Clone this repository to the Smart Reflect Server plugin folder
* [Create Youtube API Key] (https://developers.google.com/youtube/v3/getting-started)
* Add youtube API key to plugin.conf file where specified
* Enable plugin through the Smart Reflect Server plugin configuration GUI.
